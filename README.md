# Tokenly

Track your AI usage across ChatGPT, Claude, and more — all in one place.

## Status

Early prototype. Currently a Chrome extension (Manifest V3) that counts visits to supported AI tools.

**Supported:**
- ChatGPT (chatgpt.com)
- Claude (claude.ai)

## Local development

1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode** (top right)
3. Click **Load unpacked** and select this folder
4. Pin Tokenly to the toolbar

Edit files, then hit the reload icon on the extension card to see changes.
